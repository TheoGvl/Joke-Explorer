# Joke API Explorer

A sleek, modern desktop and web application built entirely in Python using the Flet framework. This app connects to the [JokeAPI](https://v2.jokeapi.dev/) to deliver a highly customizable, interactive humor experience.

## Features

* **Modern UI:** A clean, dark-themed interface inspired by Flutter, built purely with Python.
* **Smart Categories:** Choose from standard categories (Programming, Puns, Spooky) or custom mixed categories.
* **Interactive Punchlines:** For two-part jokes, the setup is displayed first, and a "Reveal Punchline" button smoothly animates the delivery into view.
* **Cross-Platform:** Can be run as a standalone desktop application or deployed as a web app with zero changes to the codebase.

## Tech Stack

* **Language:** Python 3.x
* **UI Framework:** Flet, Builds interactive web, desktop, and mobile apps in Python without prior frontend experience.
* **API Handling:** `requests` library.
* **External API:** [JokeAPI v2](https://v2.jokeapi.dev/) by Sv443.

## How to Run Locally

 **Clone or Download the Project:**
   Save the `joke_app.py` file to your local machine.

 **Install Dependencies:**
   Ensure you have Python installed, then install the required libraries using pip:
   ```
   pip install flet requests
   ```
